# Implementation Records

**Classification:** Documentation index

The Markdown files in this directory are implementation working records and historical plans unless explicitly labeled otherwise. They preserve decisions, checklists, and verification snapshots; they do not define current product behavior.

- [LIFEBOARD_5_REMAINING_EXECUTION_LEDGER.md](./LIFEBOARD_5_REMAINING_EXECUTION_LEDGER.md) is the only active completion tracker.
- [LIFEBOARD_5_DEEP_COMPLETION_TRACEABILITY.md](./LIFEBOARD_5_DEEP_COMPLETION_TRACEABILITY.md) is the release-contract evidence index.
- Current product behavior lives in the [product handbook](../product/README.md).
- Current design behavior lives in the [UI/UX guide](../design/LIFEBOARD_PRODUCT_UI_UX_GUIDE.md) and [DESIGN.md](../../DESIGN.md).

An unchecked historical item is not automatically active scope; a checked historical item is not automatically fresh release evidence.
