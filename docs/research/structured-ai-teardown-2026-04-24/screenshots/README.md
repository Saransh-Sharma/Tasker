# Screenshot Evidence

The Structured app was audited through iPhone Mirroring. Live screenshots were visible to the Computer Use tool, but local macOS screenshot export failed for the mirrored iPhone surface with:

`could not create image from display`

Because of that platform limitation, this folder does not contain PNG captures from the current pass. The teardown document and `raw-notes.md` include named observed screen states that correspond to the required screenshot list.

Recommended follow-up: capture screenshots manually from the physical iPhone or through an iOS simulator/test device where screenshots are not protected by iPhone Mirroring.
