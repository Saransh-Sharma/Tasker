# Habit Roadmap

This roadmap tracks remaining work for Habit Streaks in Tasker after the core experience shipped.

## Near Term

- Add Habit Board loading and error-state clarity.
- Add forward actions to Habit Board and Habit Library empty states.
- Surface habit mutation failures in visible Home feedback.
- Improve Dynamic Type and accessibility scaling for board-heavy surfaces.
- Add explicit close affordances and dependency-loading feedback in habit editors.
- Add habit-specific deep-link reachability from external and internal entry points.

## Mid Term

- Expand recovery cues after misses and lapses.
- Improve restart nudges so users can re-enter the habit system quickly.
- Add gentler notification batching and better reminder-window ergonomics.
- Broaden habit types once the streak contract remains stable.
- Deepen analytics for skips, recovery patterns, streak resilience, and quiet-tracking health.

## Long Term

- Add more flexible non-daily habit structures when the streak model can support them safely.
- Improve habit-task linkage without collapsing habits into tasks.
- Build more explainable simplification guidance for overloaded users.

## Non-Goals

- Social streak leaderboards
- Punitive loss-aversion mechanics
- Clinical or diagnostic framing
