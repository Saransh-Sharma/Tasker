# SwiftUI-Animations attribution

LifeBoard selectively adapts interaction ideas from [SwiftUI-Animations](https://github.com/Shubham0812/SwiftUI-Animations) by Shubham Kumar Singh. The upstream work is licensed under Apache License 2.0.

The adapted LifeBoard behavior is substantially modified to use real application state, accessibility and energy policies, local control clipping, and LifeBoard-native navigation and Journal systems. The complete license and modification notice ship in `LifeBoard/Resources/ThirdPartyNotices/SwiftUI-Animations-NOTICE.txt` and are readable from LifeBoard Settings.
