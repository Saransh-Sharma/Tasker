# LifeBoard Sun Sources

- `SunDay.source.png` is the lossless transparent source supplied as `transparent Sun1.png`.
- `SunDayPlan.source.png` is the lossless transparent source supplied as `transparent Sun2.png.png`.
- Runtime catalog variants are generated at 418, 836, and 1254 pixels for 1x, 2x, and 3x display scales.
- These assets are decorative. Product surfaces must keep text outside their bright core or apply the shared image-readability treatment.
